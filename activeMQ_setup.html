The second step is to untar the bundle of deb files.

$ tar -zxvf apache-activemq-5.15.8-bin.tar.gz

Next you need to create an activemq user and activemq group

$ sudo addgroup activemq
$ sudo adduser --ingroup activemq activemq

Next, run the following command to install each element of MySQL Community Server in consecutive order.
